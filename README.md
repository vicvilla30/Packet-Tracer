# Topología de Red – Packet Tracer

Este repositorio contiene un archivo de **Packet Tracer** (`.pkt`) que representa una topología de red diseñada para prácticas de redes y simulación.

## Contenido del repositorio

- `tu_archivo.pkt` → Archivo de Packet Tracer con la topología de red.
- `README.md` → Documentación y descripción de la topología.

## Cómo abrir el archivo

1. Descarga e instala **Cisco Packet Tracer**:  
   [Descargar desde Cisco Networking Academy](https://www.netacad.com/courses/packet-tracer)
2. Abre Packet Tracer.
3. Selecciona **Archivo → Abrir** y carga `tu_archivo.pkt`.
4. Explora los dispositivos, conexiones y configuraciones.

## Descripción de la topología

La topología incluye los siguientes dispositivos:

- **Routers:** 2  
- **Switches:** 3  
- **PCs:** 4  
- **Servidor:** 1
